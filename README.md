# 🚀 StoicLabs NFT Project

A decentralized NFT project built on the **Solana blockchain** using **Metaplex**. This project demonstrates how to create, mint, and verify NFTs with real-world metadata stored on IPFS.

---

## 🔥 Features

* 🎨 NFT Creation on Solana
* 🪙 Mint Digital Assets
* ✅ Verify NFT Collections
* 🌐 IPFS Integration (Pinata)
* ⚡ Fast & Low-cost transactions (Solana)

---

## 🛠 Tech Stack

* **Blockchain:** Solana
* **Framework:** Metaplex Token Metadata
* **Language:** TypeScript
* **Runtime:** Node.js
* **Storage:** IPFS (Pinata)

---

## 📁 Project Structure

```
new-nft/
├── create-nft.ts          # Script to create NFT
├── create-collection.ts   # Collection creation
├── verify-nft.ts          # Verify NFT in collection
├── collection.json        # Collection metadata
├── nft2.json              # NFT metadata
├── package.json
```

---

## ⚙️ Installation

```bash
git clone https://github.com/Nitin07u/Stoiclabs-NFT.git
cd Stoiclabs-NFT
npm install
```

---

## ▶️ Usage

Run scripts using:

```bash
ts-node create-collection.ts
ts-node create-nft.ts
ts-node verify-nft.ts
```
